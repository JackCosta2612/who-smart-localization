# Evaluation notes

This skill currently supports structural evaluation only.

## What can be evaluated now

- required profile sections are present;
- source inventory table is present;
- known facts table is present;
- controlled values are used for source status, confidence, and review need;
- uncertainty and human review sections are explicit.

## What cannot be evaluated yet

- correctness of country context;
- completeness of national source retrieval;
- accuracy of WHO indicator interpretation;
- suitability for a final localization decision;
- correctness of terminology mapping.

## Future evaluation set

Once real documents are approved for use, create a small benchmark with:

- one country and one health domain;
- approved source inventory;
- expert-reviewed known facts;
- expected uncertainties;
- expected handoff notes for Policy Comparison.
