# Agent behavior notes

Draft 3 demonstrates a policy-comparison handoff style while staying inside the
Country Profiling skill.

## What this example teaches

- A country profile can be useful even when many policy source classes are
  missing.
- Mark source readiness directly instead of filling gaps with assumptions.
- Frame Italy as a national policy environment with regional implementation
  layers.
- Use the handoff table as an operational checklist for the future Policy
  Comparison skill.
- State that the output is ready for scoping only when the source base is not
  ready for detailed comparison.

## What to imitate

- Conservative language.
- Strong separation between reviewed evidence, candidate sources, missing
  sources, and downstream comparison inputs.
- Repeated reminders that WHO/SMART material belongs in the next skill, not in
  the country profile unless supplied as downstream context.
- Clear review-owner assignments in the evidence-gap table.

## What not to imitate

- Do not make the profile so gap-oriented that it omits required schema
  sections.
- Do not let the handoff become a policy comparison.
- Do not use this style when the user asked for a polished narrative overview
  and supplied enough source material for one.

## Boundary before policy comparison

The draft names the comparison inputs that would be needed, including official
Italian policy text and WHO/SMART material, but it does not compare them. It
also avoids ranking gaps or judging Italian policy performance.

## Reuse by Policy Comparison

Use this draft as a task list for the next run: define the comparison question,
retrieve Italian policy documents, retrieve WHO/SMART source material, map
actors and data flows, collect regional and coverage evidence, and assign expert
review.

## Weaknesses requiring manual review

- Less narrative and less complete as a country overview than Draft 1.
- Several source rows are intentionally candidate or missing.
- The output may need expansion if chosen as the final reference example.
