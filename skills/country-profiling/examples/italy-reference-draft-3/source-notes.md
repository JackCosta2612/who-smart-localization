# Source notes

Draft 3 is source-conservative and handoff-oriented. It is intended to show how
the Country Profiling skill can be useful even when the main value is a
structured source and uncertainty map for a later Policy Comparison run.

## Reviewed or proposed sources

| Source | Why it matters |
|---|---|
| Italy: Country Health Profile 2025, OECD / European Observatory / European Commission | Current country profile for health status, accessibility, resilience, and system challenges. |
| Italy: health system review 2022 and health system summary 2022, European Observatory | Establishes the central-regional SSN model, universal coverage, financing, service delivery, and reform context. |
| PNPV 2023-2025 and Calendario nazionale vaccinale, Gazzetta Ufficiale | Official national policy source class for the first likely downstream topic. Listed as candidate because the draft does not parse the attachments. |
| PNPV 2023-2025, ISS EpiCentro | Supports high-level description of approval, publication, harmonization, and equity aims. |
| Vaccines, AIFA | Regulatory and pharmacovigilance source class for vaccines. |
| Fascicolo sanitario elettronico and PNRR digital health pages, Italian Ministry of Health | Digital health and data infrastructure context relevant to SMART localization. |
| Regional implementation documents, coverage datasets, WHO/SMART materials | Explicitly missing source classes needed before later comparison. |

## Source-use cautions

- Draft 3 intentionally uses some sources as source-class signposts rather than
  detailed evidence.
- The main output value is the handoff table and evidence-gap table.
- This draft should not be selected if the team wants the most readable broad
  country overview; it is more operational and conservative.
