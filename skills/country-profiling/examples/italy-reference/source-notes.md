# Source notes

This single Italy reference example combines the strongest parts of the three
drafts:

- the broad health-system narrative and section coverage from draft 1;
- the immunization source-family discipline from draft 2;
- the conservative handoff, uncertainty, and evidence-gap framing from draft 3.

## Endpoint decisions

Use evidence-bearing endpoints wherever possible:

| Source | Endpoint decision |
|---|---|
| Italy: Country Health Profile 2025 | Use the direct OECD `content/dam` PDF, not the OECD publication/download page. The publication page identifies the source, but the PDF is the material that must be reviewed before marking the source `Reviewed`. |
| Italy: health system review 2022 | Use the IRIS PDF bitstream or an equivalent official Observatory download, not only the Observatory landing page. |
| Italy: Health System Summary 2024 | Use the IRIS PDF bitstream for the updated concise system summary. |
| Piano Nazionale della Prevenzione 2020-2025 | Use the Ministry PDF attachment rather than only the publication page. |
| PNPV 2023-2025 and Calendario nazionale vaccinale | Use the Gazzetta full-text attachment pages for Allegato A and Allegato B. The top-level Gazzetta act page is useful for discovery but should not be the only endpoint used for detailed policy work. |
| ISS EpiCentro PNPV page | Use as a reviewed explanatory page for approval/publication and source-family context, not as a substitute for the official attachments. |
| AIFA vaccines page | Use for regulatory and pharmacovigilance context only; do not infer local supply, coverage, or implementation from it. |
| FSE material | Treat as a candidate source class until implementation details, data classes, and regional adoption evidence are retrieved and reviewed. |

## Source-use cautions

- `Reviewed` means the material endpoint was opened and used as evidence.
- Landing pages, publication pages, catalog pages, and search results should be
  marked `Candidate source` unless the linked material is also reviewed.
- Country Profiling may identify policy source classes, but it must not compare
  Italian policy against WHO guidance.
- The example intentionally avoids exact vaccine eligibility and schedule
  claims because that belongs in the future Policy Comparison pass after the
  official attachments are parsed.
- Regional implementation evidence, coverage data, digital reporting details,
  and expert validation remain required before detailed comparison.
