profiling. It is a behavior-shaping example, not a final evidence package.

## Remaining unresolved source gaps

| Source class | Status | Reason |
|---|---|---|
| Current Ministry/ISS immunization coverage dataset | Needs retrieval | Needed only when the comparison requires vaccine, dose, cohort, year, or regional uptake beyond WHO GHO national estimates. |
| Regional implementation or registry specifications | Needs retrieval | Needed when the comparison touches regional workflows, registry fields, reporting, or interoperability. |
| AIFA pharmacovigilance or vaccine safety source | Conditional retrieval | Needed only for safety monitoring, adverse event reporting, or product governance questions. |

## Source-resolution rules demonstrated

- GHO rows are retrieved values, not candidate leads.
- EuroHealthObservatory country pages are resolved to the country page and
  direct PDF URLs where available; generated example PDFs are not bundled.
- OECD is not represented as an SDMX script. OECD/EU evidence enters this
  example through the parsed Country Health Profile PDF discovered from the
  Observatory page.
- `source-leads.md` is now a short unresolved-gap list, not a long candidate
  source inventory.
- A landing page, catalog page, search page, or home page should not support a
  substantive profile claim unless its evidence-bearing text or linked source
  has been reviewed.

## Sources to retrieve before detailed Policy Comparison

- Exact PNPV/calendar attachment sections relevant to the selected comparison
  question.
- Ministry/ISS coverage datasets by vaccine, dose, cohort, region, and year if
  uptake, equity, or implementation performance is in scope.
- Regional implementation documents or registry/reporting specifications if the
  DAK comparison includes workflows or data elements.
- AIFA sources if the selected scope includes pharmacovigilance, product safety,
  or vaccine governance.
