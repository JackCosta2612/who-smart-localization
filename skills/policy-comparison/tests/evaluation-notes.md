# Evaluation notes

This repository currently supports structural evaluation only.

## What can be evaluated now

- whether the localization matrix uses the required columns;
- whether controlled values are respected;
- whether examples remain clearly marked as toy content;
- whether outputs preserve traceability fields.

## What cannot be evaluated yet

- clinical correctness;
- correctness of WHO interpretation;
- correctness of local policy interpretation;
- terminology mapping quality;
- suitability for production or deployment.

## Next evaluation step

Once real WHO and country policy excerpts are approved for use, add a small gold-standard review set with expert-reviewed expected classifications.
